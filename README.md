## Book Travel

Bem-vindo ao Book Travel, uma aplicação de planejamento de viagens desenvolvida para ajudar usuários a explorar e organizar suas viagens dos sonhos. Com esta plataforma, você pode pesquisar por lugares, países e cidades, além de descobrir pontos turísticos e salvá-los como favoritos para acessar facilmente as informações que mais lhe interessam.

### 🚧 Status do Projeto
O projeto está atualmente em desenvolvimento. Muitas funcionalidades estão em construção e atualizações serão feitas regularmente.
